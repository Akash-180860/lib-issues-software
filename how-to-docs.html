How To Instructions

These instruction assume you have cloned git@github.com:lwg/issues.git to a directory named "issues".

How to publish the issues list HTML files

Initial setup; do this once only:

cd issues
git clone git@github.com:lwg/issues.git issues-gh-pages
cd issues-gh-pages
git fetch origin
git checkout -b gh-pages origin/gh-pages

Normal update cycle:

Build fresh versions of the HTML files in the top-level issues directory, remembering to update the toc, then:

cd issues-gh-pages
copy /y ..\issues\lwg*.html .  // Windows
cp ../issues/lwg*.html .       // POSIX
git commit -a -m"your commit message here"
git push
